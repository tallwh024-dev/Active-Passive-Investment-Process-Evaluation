# Data

This project uses the course-provided `FinalProjectData.xlsx` workbook for 10 years of monthly return data.

The workbook includes:

- benchmark returns for the nine passive asset-class benchmarks
- active fund returns for corresponding actively managed funds
- risk-free rate information
- data used for active return and benchmark return analysis

The Excel workbook is not included in this public repository because it was provided as course material. To reproduce the analysis, place `FinalProjectData.xlsx` in the project root or in the `data/` folder and update the script path if needed.

Sample period:

- January 2016 through December 2025

Main benchmark tickers:

- IWB
- IWM
- IXUS
- SHV
- TLT
- IGSB
- XLRE
- GSG
- IGF
